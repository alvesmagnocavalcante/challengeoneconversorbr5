# Conversor

Este é um aplicativo Java simples que permite realizar conversões de unidades, como moedas, temperatura, quilometragem e anos-luz. O aplicativo possui uma interface gráfica e faz uso de uma API externa para obter as taxas de conversão de moedas.

## Funcionalidades

- Conversão de moedas: o aplicativo permite converter valores entre diferentes moedas, usando as taxas de conversão obtidas de uma API externa.
- Conversão de temperatura: é possível converter valores entre Celsius, Fahrenheit e Kelvin.
- Conversão de quilometragem: o aplicativo suporta a conversão entre quilômetros e milhas.
- Conversão de anos-luz: é possível converter distâncias entre anos-luz e parsecs.

## Requisitos

- Java Development Kit (JDK) 8 ou superior.
- Bibliotecas externas:
  - Swing: para a criação da interface gráfica.
  - `org.json`: para o processamento de dados JSON.

## Como usar

1. Clone este repositório em sua máquina local.
2. Certifique-se de ter o JDK instalado e configurado corretamente.
3. Compile o código-fonte do aplicativo usando o comando `javac`:
   ```
   javac Conversor.java
   ```
4. Execute o aplicativo usando o comando `java`:
   ```
   java Conversor
   ```
5. A interface gráfica do aplicativo será aberta.
6. Selecione o tipo de conversão desejado no menu suspenso "Tipo de Conversão".
7. Selecione a unidade de origem e a unidade de destino nos menus suspensos correspondentes.
8. Clique no botão "Converter" para realizar a conversão.
9. Dependendo do tipo de conversão, você pode ser solicitado a inserir um valor de entrada.
10. O resultado da conversão será exibido em uma caixa de diálogo.
11. Após a exibição do resultado, você terá a opção de realizar outra conversão ou sair do aplicativo.

## Dependências

O aplicativo faz uso das seguintes bibliotecas externas:

- Swing: biblioteca Java para criação de interfaces gráficas.
- `org.json`: biblioteca Java para processamento de dados JSON.
